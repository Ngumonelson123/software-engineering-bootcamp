# WOD - Color Changer

A simple interactive web application that generates random colors and displays them in a color box.

## Features

- **Random Color Generation**: Click a button to generate random hex colors
- **Visual Display**: Color box that changes background color dynamically  
- **Color Code Display**: Shows the current hex color code
- **Responsive Design**: Clean, centered layout that works on different screen sizes

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

```
WOD/
├── index.html      # Main HTML structure
├── style.css       # Styling and layout
├── scripts.js      # JavaScript functionality
└── README.MD       # Project documentation
```

## How to Run

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. Click the "Change Color" button to generate random colors

## Code Overview

### JavaScript Functionality
- Generates random hex colors using `Math.random()`
- Updates DOM elements dynamically
- Event-driven color changes on button click

### CSS Features
- Centered container layout
- Rounded corners and hover effects
- Clean typography with Arial font family
